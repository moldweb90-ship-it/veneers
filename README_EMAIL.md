# Настройка отправки email

## Переменные окружения в Vercel

Для работы отправки email нужно настроить переменные окружения в Vercel:

1. Перейдите в настройки проекта в Vercel
2. Откройте вкладку "Environment Variables"
3. Добавьте следующие переменные:

```
EMAIL_USER=ваш_email@mail.ru
EMAIL_PASSWORD=ваш_пароль_приложения_mail_ru
```

## Получение пароля приложения Mail.ru

1. Войдите в аккаунт mail.ru
2. Перейдите в настройки безопасности
3. Создайте "Пароль приложения" для почтового клиента
4. Используйте этот пароль в переменной окружения `EMAIL_PASSWORD`

## Важно

- Никогда не коммитьте пароли в git
- Используйте только переменные окружения в Vercel
- После добавления переменных окружения, пересоберите проект
