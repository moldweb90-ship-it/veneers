# LIFE DENTAL - Landing Page

Лендинг для стоматологической клиники LIFE DENTAL (Виниры Кишинев).

## Технологии

- HTML5
- Tailwind CSS (CDN)
- Font Awesome (CDN)
- Vanilla JavaScript

## Локальная разработка

### Установка зависимостей (опционально)

```bash
npm install
```

### Запуск локального сервера

```bash
npm start
```

Или без npm:

```bash
npx http-server . -p 5000 -o
```

Сайт будет доступен по адресу: http://localhost:5000

## Деплой на Vercel

1. Создайте репозиторий на GitHub
2. Загрузите код в репозиторий
3. Подключите репозиторий к Vercel
4. Vercel автоматически определит настройки из `vercel.json`

Или через CLI:

```bash
npm i -g vercel
vercel
```
