# Руководство по URL для рекламных кампаний

## Форматы URL для разных языков

### Для рекламы на румынском языке:
```
https://yourdomain.com?ro
```
или
```
https://yourdomain.com?lang=ro
```

### Для рекламы на английском языке:
```
https://yourdomain.com?en
```
или
```
https://yourdomain.com?lang=en
```

### Для рекламы на русском языке (по умолчанию):
```
https://yourdomain.com
```
или
```
https://yourdomain.com?ru
```
или
```
https://yourdomain.com?lang=ru
```

## Примеры использования в рекламных кампаниях

### Google Ads / Facebook Ads / Instagram Ads

**Румынская кампания:**
- URL: `https://yourdomain.com?ro`

**Английская кампания:**
- URL: `https://yourdomain.com?en`

**Русская кампания (основная):**
- URL: `https://yourdomain.com` (без параметров, открывается на русском)

## Как это работает

1. При переходе по ссылке с параметром `?ro` или `?en`, сайт автоматически переключается на нужный язык
2. Если параметр не указан, сайт открывается на русском языке (по умолчанию)
3. Пользователь может вручную переключить язык через переключатель в шапке сайта

## Рекомендации

- Используйте короткий формат `?ro` и `?en` - он более удобен для рекламы
- Всегда тестируйте ссылки перед запуском рекламы
- Можно использовать оба формата одновременно (поддерживается обратная совместимость)
